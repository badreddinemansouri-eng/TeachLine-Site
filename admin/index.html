Proxy Cloudflare Worker (recommandé)

Déployez le fichier worker.js sur Cloudflare Workers pour garder votre token GitHub invisible côté public. Laissez vide pour utiliser le token direct.

Ouvrir Cloudflare
github.com → Settings → Developer settings → Tokens (classic) → New → cochez repo → Generate.

Mise en ligne GitHub Pages (gratuit)

1
Créez un dépôt public

github.com → New repository → Nom: TeachLine-Site → Public → Create. Créez les dossiers: content/posts/, content/resumes/, content/packs/, content/files/, access/

2
Uploadez les fichiers du site

Add file → Upload → Glissez index.html et le dossier admin/. Commit.

3
Settings → Pages → main → Save ✅

Votre site sera live sur: username.github.io/TeachLine-Site/
