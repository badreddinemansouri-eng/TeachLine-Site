Paramètres du dépôt

Pourquoi GitHub ?

Votre site est hébergé sur GitHub Pages (100% gratuit). Le contenu est stocké dans votre dépôt sous forme de fichiers. Ce panneau configure l'accès pour que l'admin puisse y écrire directement, sans que vous touchiez au code.

Créez votre token sur : github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Cochez repo → Generate token. Copiez-le immédiatement, il n'est affiché qu'une fois.

Guide : Mettre le site en ligne

1

Créez un compte GitHub

Allez sur github.com → Sign up. Gratuit, rapide.

2

Créez un dépôt public

New repository → Nom : TeachLine-Site → Public → Create. Ensuite créez les dossiers : content/posts/ content/resumes/ content/files/ (créez un fichier .gitkeep vide dans chacun).

3

Uploadez les fichiers du site

Add file → Upload files → Glissez index.html et le dossier admin/ complet. Commit.

4

Activez GitHub Pages

Settings → Pages → Source : main → Dossier : / (root) → Save. Votre site est en ligne sur votre-username.github.io/TeachLine-Site/ ✅

5

Créez votre token et configurez l'admin

Settings → Developer settings → Personal access tokens → Tokens (classic) → New token → cochez repo → Copiez et collez dans la configuration ci-dessus. Vous pouvez maintenant publier en un clic ! 🎉
