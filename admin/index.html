Accès par contenu

Emails autorisés spécifiquement dans chaque publication

Pour définir des accès spécifiques à une publication ou ressource, editez-la et ajoutez les emails dans le champ « Emails autorisés ». Ces accès sont enregistrés directement dans le fichier de ce contenu sur GitHub.

Chargement des contenus privés…

Fonctionnement du système d'accès

Étape 1 : L'élève vous contacte (WhatsApp ou email) pour demander l'accès à un cours privé.

Étape 2 : Vous ajoutez son email ici (accès global) ou dans le contenu spécifique. Cliquez Sauvegarder.

Étape 3 : L'élève va sur le site → clique « Accès élève » → entre son email → il voit instantanément les contenus débloqués.

Configuration GitHub

Nécessaire pour la publication automatique

Dépôt GitHub

github.com → Settings → Developer settings → Personal access tokens (classic) → New token → cochez repo.

Guide de déploiement GitHub Pages

1
Créez un compte GitHub

Allez sur github.com → Sign up. Gratuit.

2
Créez un dépôt public

New repository → Nom : TeachLine-Site → Public → Create. Créez les dossiers content/posts/, content/resumes/, content/files/, access/.

3
Uploadez les fichiers du site

Add file → Upload → Glissez index.html et le dossier admin/. Commit.

4
Activez GitHub Pages ✅

Settings → Pages → Source : main → root → Save. Votre site sera sur : votre-username.github.io/TeachLine-Site/
